# Sydney Sanierung Redesign

Statische Website für Sydney Sanierung in Düsseldorf.

## Seiten

- `index.html`: Startseite
- `projekte/projekt-stockum/index.html`: Projektgeschichte und Galerie mit 31 Bildern
- `impressum.html`, `datenschutz.html`, `404.html`

## Lokal starten

```powershell
python -m http.server 4174
```

Danach `http://127.0.0.1:4174` öffnen. Kein Build-Schritt und keine Paketinstallation erforderlich.
